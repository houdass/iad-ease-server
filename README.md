# iad-ease-server
**A restful API using :**
- _Node.js_
- _Express_
- _MongoDB_
- _Mongoose_
- _Gulp_
_ _Nodemon_
- _Socket.io_
- _Passport.js_
- _JsonWebToken_
- _Mocha_
- _Should_
- _Sinon_


**Don't forget, you should run these commands :**

_sudo npm install apidoc -g_


**Available commands :**

- _npm start :_ Run the project
- _npm test :_ Run tests
- _npm apidoc :_ Generate the api doc
